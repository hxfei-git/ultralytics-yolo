# 训练任务队列

本目录用于存放待训练的任务配置文件，供训练调度器自动执行。

## 如何提交训练任务

1. 将你的模型配置文件（`.yaml`）复制到此目录。
2. **文件名必须为 `{实验名称}.yaml` 格式**  
   例如：`yolo11n-FED.yaml`、`ablation_v3.yaml`
3. 调度器会自动：
   - 使用该文件作为 `--yaml` 参数
   - 使用文件名（不含 `.yaml` 后缀）作为 `--name` 参数
4. 训练成功后，任务文件会被自动删除。

## 注意事项

- 每次调度器运行只处理一个任务（按文件名字典序优先）。
- 如果训练失败，任务文件会保留以便排查问题。
- 请勿在任务运行时修改或删除正在处理的文件。
- 本目录需保留在 Git 中，以维持项目结构和文档完整性。
